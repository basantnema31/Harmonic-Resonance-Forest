# Explainable Resonance Visualization Dashboard

## Overview

This dashboard provides explainability tools for the
Harmonic Resonance Forest (HRF) framework.

## Features

### 1. Class Resonance Visualization

Displays class-wise resonance contributions
for individual predictions.

### 2. Resonance Heatmap

Shows resonance intensity patterns
across samples and classes.

### 3. Resonance Comparison

Provides resonance heatmaps with
prediction analysis.

### 4. Decision Boundary Visualization

Visualizes classification regions
using PCA projection.

### 5. Evolution Analytics

Tracks:

- Accuracy progression
- Gamma evolution
- Frequency evolution

python -m streamlit run visualizations/resonance_dashboard.py
